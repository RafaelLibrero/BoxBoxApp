## ✨ Características

- 🔄 Arquitectura MVVM modular y escalable
- 🗄️ Repositorios y casos de uso desacoplados
- 🌐 Retrofit + Coroutines para llamadas de red
- 📦 Inyección de dependencias con Hilt
- 🎨 Vistas diseñadas en XML
- ✅ ViewModel y StateFlow
